# Mini Stadium Rental System ⚽🏟️

Mini Stadium Rental System is a desktop application developed with **C# Windows Forms** and **SQL Server**.
It helps manage mini football stadium reservations, customers, payments, and schedules in an organized way.

---

## 🚀 Features

### 🏟️ Stadium Management

* Add new stadiums
* Update stadium information
* Delete stadiums
* View active stadium list

### 👤 Customer Management

* Add customers
* Update customer details
* Delete customers
* Customer search

### 📅 Reservation Management

* Book stadium by date and time
* Prevent duplicate reservations
* Calculate total price automatically
* View reservation history

### 💳 Payment Management

* Record payments
* Track paid/unpaid reservations
* Payment reports

### ⏰ Schedule Management

* Daily stadium schedule
* Available / reserved hours view

### 🔐 Security

* Authentication system
* Role-based access (Admin / User)

---

## 🛠️ Technologies Used

* C#
* Windows Forms
* SQL Server
* ADO.NET
* Visual Studio

---

## 📂 Database Tables

* Stadiums
* Customers
* Reservations
* Payments
* Users

---

## ▶️ How to Run

1. Clone this repository
2. Open project in Visual Studio
3. Create database in SQL Server
4. Update connection string
5. Run the project

This project was created to digitalize and simplify the mini stadium rental process.
